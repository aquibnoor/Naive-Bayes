# Naive Bayes Classification

## Overview

This project applies the Naive Bayes algorithm to perform supervised classification on structured data. It demonstrates preprocessing, probability-based prediction, and model evaluation.

---

## Objective

Develop a lightweight probabilistic classifier capable of accurately predicting the target variable using Bayes' theorem.

- Scikit-learn
